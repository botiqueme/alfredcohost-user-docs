# Ajouter du contenu à la library de la propriété

Cette page explique comment remplir la Property Library avec les informations qu’Alfred utilise pour répondre aux voyageurs. Pour le concept : [Qu’est-ce que la Property Library ?](/fr/concepts/libraries_c.md)

Pour accéder à la Library d’une propriété : **My Properties** > <img src="../media/library-icon.png" alt="book icon" style="height: 1em; vertical-align: middle;"> sur la propriété > **Item Category** > **Subcategory**.

## Remplir un item textuel

Certains items (comme *WiFi name*, *door code*, *checkout time*) ont déjà un champ dédié. Saisissez la valeur dans le **champ texte** fourni.

> **TIP** Plus de conseils sur les voci de library : [Conseils d’écriture](/fr/procedures/writing_tips.md).

## Ajouter un custom text item

Pour ajouter une information non couverte par les champs par défaut :

1. Cliquez sur **Add custom info**.
2. Choisissez un **friendly label** pour identifier l’item.
3. Ajoutez une **short description** pour préciser de quoi il s’agit.
4. Réglez **Info type** sur `Text`.
5. Saisissez le contenu et cliquez **Save**.

> **TIP** Plus de conseils : [Conseils d’écriture](/fr/procedures/writing_tips.md).

## Ajouter un custom media item

Pour envoyer une image ou une vidéo :

1. Cliquez sur **Add custom info**.
2. Saisissez un **label** et une **short description**.
3. Réglez **Info type** sur `Image` ou `Video`.
4. Uploadez le fichier et cliquez **Save**.
