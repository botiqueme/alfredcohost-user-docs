# Changer la carte de paiement

Dans le portail Alfred Co.Host, vous pouvez associer une carte de paiement à votre profil. Cette carte est utilisée pour **tous les paiements Alfred Co.Host**.

Pour voir ou modifier la carte : **My host profile** > **My saved card**, puis enregistrez les nouvelles données de carte.

> **IMPORTANT**
> - Après l’ajout d’une nouvelle carte, la mise à jour peut prendre quelques minutes.
> - **Toute nouvelle carte prend effet à partir du prochain billing cycle**, quel que soit le cycle actuel.
