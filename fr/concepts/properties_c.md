# À propos des propriétés

Les properties sur la plateforme Alfred reflètent vos unités louables réelles — c’est-à-dire les unités que les voyageurs peuvent réserver, par exemple :
- une maison ou un appartement entier
- une chambre dans un hôtel, B&B ou auberge
- une chambre dans une guesthouse avec espaces communs

Lors d’un contact, **Alfred identifiera le voyageur dans la bonne property** et **fournira des réponses basées uniquement sur cette property**.
