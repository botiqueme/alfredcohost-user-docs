# Le proprietà

Le property nella piattaforma Alfred riflettono le tue unità reali in affitto — cioè unità che gli ospiti possono prenotare, ad esempio:
- una casa o un appartamento intero
- una stanza in hotel, B&B o ostello
- una stanza in una guesthouse con spazi in comune

Quando un ospite contatta Alfred, **Alfred lo associa alla property corretta** e **fornisce risposte basate solo su quella property**.
