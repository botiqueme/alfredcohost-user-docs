# Aggiungere contenuti alla library della proprietà

Questa pagina spiega come compilare la Property Library con le informazioni che Alfred usa per rispondere agli ospiti. Per il concetto: [Cos’è la Property Library?](/it/concepts/libraries_c.md)

Per accedere alla Library di una proprietà: **My Properties** > <img src="../media/library-icon.png" alt="book icon" style="height: 1em; vertical-align: middle;"> sulla proprietà > **Item Category** > **Subcategory**.

## Compilare un item testuale

Alcuni item (come *WiFi name*, *door code*, *checkout time*) hanno già un campo dedicato. Inserisci il valore nel **campo testo** fornito.

> **TIP** Altri consigli sulle voci di library: [Consigli di scrittura](/it/procedures/writing_tips.md).

## Aggiungere un custom text item

Per aggiungere qualcosa non coperto dagli item predefiniti:

1. Clicca **Add custom info**.
2. Scegli un **friendly label** che aiuti a riconoscere lo scopo dell’item.
3. Aggiungi una **short description** per chiarire di cosa si tratta.
4. Imposta **Info type** su `Text`.
5. Inserisci il contenuto e clicca **Save**.

> **TIP** Altri consigli: [Consigli di scrittura](/it/procedures/writing_tips.md).

## Aggiungere un custom media item

Per caricare un’immagine o un video:

1. Clicca **Add custom info**.
2. Inserisci un **label** e una **short description**.
3. Imposta **Info type** su `Image` o `Video`.
4. Carica il file e clicca **Save**.
