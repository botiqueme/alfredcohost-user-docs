# Test assistant

Antes de publicar puedes probar cómo responde Alfred a las preguntas de los huéspedes usando el contenido de la library de la propiedad. Así puedes ver cómo funcionan tus datos en la práctica y asegurarte de que todo esté bien configurado.

## Cómo probar Alfred

Para probar las respuestas de Alfred para una propiedad concreta:

1. Ve a **My Properties**.
2. Haz clic en el icono <img src="../media/test-icon.png" alt="chat message icon" style="height: 1em; vertical-align: middle;"> junto a la propiedad que quieras probar.
3. Se abre una ventana de prueba donde puedes interactuar con una versión de prueba de Alfred.

El test assistant usa la información guardada en la library de esa propiedad, así que verás exactamente cómo responderá Alfred a tus huéspedes.

> **NOTE** Funciones interactivas como mensajes de voz o handoff humano no están soportadas en el chat de prueba. La versión de prueba sirve para comprobar cómo Alfred usa el contenido de tu library para responder.

## Qué probar

Prueba a hacer preguntas que tus huéspedes podrían hacer, por ejemplo:
- Cómo hacer el check-in
- Dónde encontrar la contraseña Wi-Fi
- Cómo usar electrodomésticos o sistemas de la propiedad
- Recomendaciones locales
- Cualquier información personalizada que hayas añadido

Así compruebas que Alfred encuentra y usa correctamente el contenido de la library antes de que los huéspedes la usen.

> **TIP** Si Alfred no responde como esperas, revisa que los contenidos de la library sean claros, concretos y redactados desde el punto de vista del huésped. Consulta nuestros [consejos de escritura](/es/procedures/writing_tips.md).
