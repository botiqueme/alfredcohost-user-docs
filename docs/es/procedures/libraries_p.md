# Añadir contenido a la library de la propiedad

Esta página explica cómo rellenar la Property Library con la información que Alfred usa para responder a los huéspedes. Para el concepto: [¿Qué es la Property Library?](/es/concepts/libraries_c.md)

Para acceder a la Library de una propiedad: **My Properties** > <img src="../media/library-icon.png" alt="book icon" style="height: 1em; vertical-align: middle;"> en la propiedad > **Item Category** > **Subcategory**.

## Rellenar un item textual

Algunos items (como *WiFi name*, *door code*, *checkout time*) ya tienen un campo dedicado. Introduce el valor en el **campo de texto** proporcionado.

> **TIP** Más orientación sobre contenidos de library: [Consejos de escritura](/es/procedures/writing_tips.md).

## Añadir un custom text item

Para añadir algo que no cubren los items por defecto:

1. Haz clic en **Add custom info**.
2. Elige un **friendly label** que ayude a reconocer el propósito del item.
3. Añade una **short description** para aclarar de qué trata.
4. Pon **Info type** en `Text`.
5. Introduce el contenido y haz clic en **Save**.

> **TIP** Más orientación: [Consejos de escritura](/es/procedures/writing_tips.md).

## Añadir un custom media item

Para subir una imagen o un vídeo:

1. Haz clic en **Add custom info**.
2. Introduce un **label** y una **short description**.
3. Pon **Info type** en `Image` o `Video`.
4. Sube el archivo y haz clic en **Save**.
