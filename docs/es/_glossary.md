## Property

Unidad que gestionas en Alfred Co.Host (ej. un apartamento o habitación). Cada property tiene su propia library y subscription.

## Library

El conjunto de contenidos (texto, medios, políticas) que Alfred usa para responder a los huéspedes de una property concreta.

## Hands off

Cuando Alfred pasa una conversación a un agente humano (ej. cuando un huésped reporta un problema o pide ayuda que Alfred no puede ofrecer).
