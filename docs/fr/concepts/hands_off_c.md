# Hands off

Lorsque vous abonnez une propriété, vous devez fournir deux numéros : votre **Phone number** et votre **Hands off number**. Comprendre la différence entre les deux vous aide à configurer Alfred correctement.

## Phone number vs Hands off number

Le **Phone number** est votre numéro de contact principal. Il sert aux communications générales (compte et facturation).

Le **Hands off number** est le numéro auquel vous ou votre équipe recevrez les notifications lorsqu’un voyageur a besoin d’aide humaine. Quand Alfred transfère une conversation, les notifications sont envoyées à ce numéro.

> **IMPORTANT** Il existe aussi un **Guest contact number** — défini par Alfred Hospitality AI — que vous partagez avec vos voyageurs pour qu’ils puissent contacter Alfred via WhatsApp, Telegram ou d’autres canaux supportés. Vous le voyez dans **My Properties** > <img src="../media/touchpoints.png" alt="communication channels icon" style="height: 1em; vertical-align: middle;"> à côté de chaque propriété.

## Comment fonctionne le Hands off

Alfred gère la plupart des questions des voyageurs automatiquement grâce à la library de la propriété. Certaines situations nécessitent toutefois une intervention humaine — par exemple lorsqu’un voyageur signale un problème, demande un service particulier ou a besoin d’une aide qui dépasse ce qu’Alfred peut fournir.

Dans ce cas, Alfred **transfère** la conversation en :
1. Vous notifiant (ou votre équipe) sur le Hands off number
2. Passant la conversation à un agent humain
3. Fournissant le contexte sur ce dont le voyageur a besoin

Ainsi les voyageurs obtiennent toujours l’aide nécessaire, que ce soit via Alfred ou via votre équipe.

## Quand le Hands off se déclenche

Le Hands off est déclenché lorsque :
- Un voyageur signale un problème
- Un voyageur demande des services qui nécessitent une coordination humaine
- La conversation nécessite une attention personnelle au-delà des réponses automatiques d’Alfred
- Vos policies indiquent que certaines situations doivent être escaladées

Vous pouvez contrôler quand le Hands off se déclenche en configurant des [policies](/fr/procedures/writing_policies.md) qui indiquent à Alfred quand transférer les conversations.
