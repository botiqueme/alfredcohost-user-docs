# Test assistant

Avant la mise en ligne, vous pouvez tester comment Alfred répond aux questions des voyageurs en vous appuyant sur le contenu de la library de la propriété. Cela vous permet de vérifier en pratique que les informations fonctionnent et que tout est bien configuré.

## Comment tester Alfred

Pour tester les réponses d'Alfred pour une propriété donnée :

1. Allez dans **My Properties**.
2. Cliquez sur l’icône <img src="../media/test-icon.png" alt="chat message icon" style="height: 1em; vertical-align: middle;"> à côté de la propriété à tester.
3. Une fenêtre de test s’ouvre où vous pouvez interagir avec une version test d’Alfred.

Le test assistant utilise les informations enregistrées dans la library de cette propriété, vous voyez donc exactement comment Alfred répondra à vos voyageurs.

> **NOTE** Les fonctionnalités interactives comme les messages vocaux ou le handoff humain ne sont pas disponibles dans le chat de test. La version test sert à vérifier comment Alfred utilise le contenu de votre library pour répondre.

## Que tester

Posez des questions que vos voyageurs pourraient poser, par exemple :
- Comment faire le check-in
- Où trouver le mot de passe Wi-Fi
- Comment utiliser les équipements ou systèmes de la propriété
- Recommandations locales
- Toute information personnalisée que vous avez ajoutée

Cela vous aide à vérifier qu’Alfred trouve et utilise correctement le contenu de la library avant que les voyageurs ne l’utilisent.

> **TIP** Si Alfred ne répond pas comme prévu, vérifiez que les voci de la library sont claires, précises et rédigées du point de vue du voyageur. Consultez nos [conseils d’écriture](/fr/procedures/writing_tips.md).
