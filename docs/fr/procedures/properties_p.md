# Travailler avec les propriétés

Cette page explique comment travailler avec les properties dans la plateforme. Si vous ne savez pas comment Alfred utilise les properties, consultez d’abord le concept : [Qu’est-ce qu’une property ?](/fr/concepts/properties_c.md)

## Créer une propriété

Pour créer une propriété, allez dans **Manage my properties** > **Add a property**. Saisissez un nom reconnaissable pour la propriété et cliquez sur **Save**.

> **TIP** Pour des chambres dans la même propriété, utilisez une convention de nommage claire et cohérente — par exemple comme vous la désignez au quotidien.  
> Exemple : **Old Town Rooms – Room A** si **Room A** est ce que vous et votre équipe utilisez avec les voyageurs.

## Supprimer une propriété

> **WARNING** La suppression d’une propriété est irréversible !

Pour supprimer une propriété, allez dans **Manage my properties**, trouvez la propriété à supprimer et cliquez sur l’icône <img src="../media/delete-icon.png" alt="trash icon" style="height: 1em; vertical-align: middle;"> :
- **Si la propriété n’a pas de subscription active**, elle sera supprimée définitivement après quelques minutes.
- **Si la propriété a une subscription active**, la suppression de la propriété supprimera aussi la subscription. La propriété sera définitivement supprimée **à la fin du billing cycle**. Vérifiez le billing cycle dans **My Host Profile** > **My subscriptions**.
